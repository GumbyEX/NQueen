Genetic and Simulated Annealing

In order to compile and run, CD into the folder(Grillo_Salvatore_4200p2) Main.java is in, and type "javac Main.java" into the terminal.
From there, type "java Main" and the program should begin to run. 
By default, it is configured to do 1 iteration and print out the board.
You can change this by changing the "Final Int numIterations" near the top of the file.
Additionally, you can change parameters such as number of queens.
